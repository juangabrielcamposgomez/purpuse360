# 📖 Glosario Técnico: Purpose360 AI
**Definición de Términos Clave del Proyecto**  
**Rol:** Senior GenAI Developer  
**Estado:** Finalizado

Este glosario sirve como referencia para desarrolladores, stakeholders y nuevos miembros del equipo para asegurar una terminología común.

---

### A
*   **A2UI (Agent-to-User Interface):** Concepto de interfaces generadas dinámicamente por un agente de IA para el usuario.
*   **AG-UI:** Protocolo de comunicación que permite al agente manipular componentes visuales y el estado del frontend.
*   **Agentic Canvas:** El espacio de trabajo visual donde el agente puede "pintar" o mover elementos interactivos.

### C
*   **Checkpoints (LangGraph):** Mecanismo para guardar el estado de un hilo de conversación, permitiendo la persistencia y la capacidad de reanudar flujos complejos.
*   **CopilotKit:** Framework utilizado para integrar capacidades de IA y componentes de UI generativa en aplicaciones React.

### D
*   **Durable Threads:** Hilos de conversación que persisten en una base de datos (Postgres), evitando la pérdida de contexto entre sesiones del navegador.

### G
*   **Generative UI:** Interfaz de usuario que se crea o modifica en tiempo real por un modelo de lenguaje en lugar de estar pre-programada de forma estática.

### H
*   **HITL (Human-in-the-Loop):** Patrón de diseño donde el agente de IA requiere la aprobación o intervención humana antes de realizar acciones críticas.

### L
*   **LangGraph:** Librería para construir agentes de IA basados en grafos cíclicos dirigidos, permitiendo flujos de razonamiento complejos y memoria.
*   **Lead Store:** Capa de abstracción que gestiona la persistencia de los leads, pudiendo alternar entre Notion y un archivo JSON local.

### M
*   **MCP (Model Context Protocol):** Protocolo abierto para conectar modelos de IA con fuentes de datos externas y herramientas (ej: servidores de Notion, Slack, etc.).
